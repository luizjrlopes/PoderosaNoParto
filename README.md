# Poderosa no Parto

Este repositório contém o protótipo web desenvolvido em React para auxiliar gestantes no preparo para o parto.

## Estrutura de diretórios

- `src/`: código-fonte principal do aplicativo React.
- `public/`: arquivos estáticos servidos diretamente.
- `docs/`: documentação funcional e técnica.
- `notebooks/`: espaço reservado para experimentos exploratórios.
- `models/`: armazenamento de artefatos treinados.
- `scripts/`: utilidades de automação e deploy.
- `tests/`: cenários e utilitários de teste.

Para uma visão completa da proposta do aplicativo, consulte a documentação principal em `docs/project-overview.md`.
