# Código-fonte

Esta pasta contém o aplicativo React completo.

## Organização sugerida
- `componentes/`: componentes compartilhados reutilizáveis.
- `views/`: telas e fluxos principais.
- `routes/`: definição de rotas e navegação.
- `services/`: integrações com APIs e utilidades de dados.
- `styles/` e `assets/`: estilos globais e recursos estáticos (imagens, áudios, ícones).
- `utils/`: funções auxiliares.

Mantenha novos módulos dentro de `src/` e siga as convenções descritas no `AGENTS.md` raiz.
