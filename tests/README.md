# Testes

Use esta pasta para organizar testes automatizados, dados de fixtures e utilidades de mocking. Adote a mesma estrutura do código-fonte (`unit`, `integration`, `e2e`, etc.) para facilitar a manutenção.

## Testes Python

Após instalar as dependências listadas em `requirements.txt` e ativar a `.venv`, execute:

```bash
pytest
```

O arquivo `test_powerbirth.py` demonstra como validar os módulos sintéticos de IA localizados em `python_src/powerbirth/`.
